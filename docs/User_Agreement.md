## 一、软件说明

1.1 SeekU 是一款基于 Flutter 框架开发的课表管理应用，面向重庆大学学生的学习场景设计。本软件当前以 Windows 桌面端为主，后续将逐步扩展至 Android 等平台。

1.2 本软件提供以下核心功能：
- 多学期课程表的本地管理与展示（周视图、日视图）；
- 课程的手动新增、编辑、查看与删除；
- Excel（.xlsx、.xls）/ CSV 等格式的课表导入与预览确认；
- 基于 AI 大模型的 PDF 及图片课表识别导入（实验性功能）；
- 教室校区自动推断；
- CQU-Openlib 自动资源匹配，在课程详情页展示相关资源外部链接；
- 本地数据持久化与离线查看。

1.3 本软件**不是**重庆大学官方产品，系独立开发的开源工具。本软件与重庆大学及其教务系统无任何关联。

1.4 本软件当前版本为早期正式版本（v0.1.1），功能仍在持续迭代中，可能存在不完善之处。用户在使用时应充分知悉此情况。

## 四、数据与隐私

4.1 **数据本地化存储**：本软件的核心课表数据、学期信息、节次设置等均存储于用户设备的本地数据库（SQLite）中。在未接入账号系统与云同步功能之前，我们不会主动收集、上传或存储您的课程数据。

4.2 **AI API 密钥**：您自行配置的 Moonshot / Kimi API Key 当前以明文形式保存在本地偏好设置（SharedPreferences）中，仅用于向对应 AI 服务提供商发起请求。请注意，此种存储方式面向开发期便利性，后续版本将迁移至更安全的本地密钥存储方案。

4.3 **第三方 API 调用**：当您使用 AI 识别导入功能时，您上传的课表文件（PDF、图片）及课程文本内容将被发送至 Moonshot / Kimi 等第三方 AI 服务提供商的服务器。这部分数据的传输和处理受该第三方服务提供商的隐私政策约束，我们不对第三方服务提供商的数据处理行为承担责任。建议您在使用前阅读相关第三方服务提供商（如[月之暗面 Moonshot](https://platform.moonshot.cn/)）的隐私政策。

4.4 **网络请求**：本软件当前版本可能发起以下网络请求：AI 识别功能所需的 Moonshot / Kimi 等第三方 API 调用；课程详情页 CQU-Openlib 自动资源匹配所需的公开搜索索引请求。Openlib 功能仅用于匹配公开课程资源页面，SeekU 只缓存资源标题、摘要、链接、匹配分数与缓存时间等元数据，不默认下载或搬运资源文件。后续版本若增加教务网页导入、云同步等功能，本协议将同步更新以说明对应的数据处理方式。
